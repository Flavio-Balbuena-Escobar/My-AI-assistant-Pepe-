# Pepe AI, my Personal Proyect

Pepe AI is a desktop personal assistant developed in Python.

## Features

- Voice recognition
- Text-to-speech
- WhatsApp messaging
- Web search
- Wikipedia search
- Custom chatbot powered by ChatterBot
- Application launcher

## Technologies

- Python
- CustomTkinter
- ChatterBot
- SQLite
- SpeechRecognition
- pyttsx3

## Installation

```bash
pip install -r requirements.txt
python -m spacy download en_core_web_sm
```

## Usage

```bash
python pepe_ai_main.py
```

## About this project

This project was developed in 2025, at a time when I was still very inexperienced in programming. Despite that, I decided to take on the challenge and build something that would push me beyond what I thought I was capable of creating.

What started as a school project soon grew far beyond the expectations of a regular curricular assignment. As the project evolved, it was selected to participate in a competition organized by my school alongside other IB schools, becoming one of the most meaningful experiences of my academic journey.

More importantly, this project became much more than a competition entry. It became an opportunity to learn, experiment, make mistakes, and improve. Through its development, I discovered a genuine interest in artificial intelligence and machine learning, while also gaining my first insights into another field that would later capture my attention: cybersecurity.

Looking back, Pepe AI represents more than lines of code. It represents curiosity, perseverance, and the beginning of a path of continuous learning in technology and computer science.
